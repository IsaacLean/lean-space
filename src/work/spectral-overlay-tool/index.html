Spectral Overlay Tool

  • View Live
  • Case Study

Description

The spectral overlay tool is a web application that visualizes the absorption and emission spectra for common fluorescent dyes and quenchers. Users can overlay them as multiplexed sets according to recommended combinations for each qPCR device. Supports use on desktop and mobile screens.
